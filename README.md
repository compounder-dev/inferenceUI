# InferenceUI

An open-source component library for inference platforms — compute markets, runtime control planes, and inference operating systems.

Built on top of **shadcn/ui**. Themed for high-density, mission-critical surfaces. MIT licensed.

```bash
bun install
bun dev
```

## Stack

- Next.js 16, React 19, TypeScript
- Tailwind v4, shadcn/ui (vendored)
- Geist Sans (body) · Geist Mono (data) · Big Shoulders (display)
- Accessible by default — every primitive ships with Radix-grade keyboard, screen-reader, and focus support

## Status

`v0.1` — Phase 0. The instrument hero pattern is locked. Component library and registry roll out across subsequent phases.
